# sensor-data-visualizer
Real-time sensor data visualization with matplotlib and multiple theme support

## Key Features

- Built high-performance live data visualization toolkit in Python with sub-50ms refresh rates and 60-point sliding window buffers for multiple concurrent streams.

- Implemented robust error handling and configurable update speeds (0.01-0.1s) for sensor monitoring and telemetry analysis applications.

- Developed interactive dashboard with dynamic theming engine featuring pre-built visualization themes and real-time configuration controls for customizing chart types, colors, and update intervals.
